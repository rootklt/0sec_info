## 零组安全资料库爬虫

---

​    这几天看了一些x组安全的资料，觉得很不错，资料也比较全面，为了方便离线时能查阅，写了一个简单的爬虫把资料爬下来。这个脚本可以爬gitbook类的资料。

### 依赖

---

1. requests
2. bs4
3. lxml

### 例子

```bash
python3 0sec_info.py
```
